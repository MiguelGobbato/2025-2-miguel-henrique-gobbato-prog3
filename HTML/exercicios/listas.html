Redes de Computadores

IP
Um endereço IP (Internet Protocol) é um identificador numérico exclusivo atribuído a cada dispositivo conectado a uma rede de computadores que utiliza o protocolo IP para comunicação. É como um endereço postal para a internet, permitindo que os dados sejam enviados e recebidos corretamente entre dispositivos.
DNS
DNS, ou Sistema de Nomes de Domínio, é o sistema que traduz nomes de domínio legíveis por humanos (como www.google.com) em endereços IP numéricos que os computadores e dispositivos usam para se comunicar na internet. É frequentemente comparado à lista telefónica da internet, pois permite que os utilizadores acedam a recursos online digitando nomes amigáveis em vez de ter de memorizar os complexos endereços IP de cada site. www.exemplo.com
Roteador
Um roteador é um dispositivo que conecta duas ou mais redes e encaminha pacotes de dados entre elas, atuando como um "direcionador" de informações na internet e em redes locais (LANs). Ele é essencial para distribuir a conexão de internet para diversos aparelhos (como computadores e smartphones), permite a comunicação entre dispositivos na rede e oferece recursos de segurança, como firewalls.
