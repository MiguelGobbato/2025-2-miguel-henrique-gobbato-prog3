O que é o HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar o conteúdo de páginas web. Não é uma linguagem de programação, mas sim uma forma de organizar e apresentar informações na internet, como texto, imagens e vídeos, para que os navegadores possam interpretá-las. Pesquisa efetuada em 15/08/2025.
