IP
IP é a sigla para Protocolo de Internet (Internet Protocol), um conjunto de regras que definem como os dados são enviados e recebidos na internet, funcionando também como um endereço numérico exclusivo (Endereço IP) que identifica cada dispositivo numa rede. Pense no endereço IP como o endereço da sua casa na internet: ele permite que os dados cheguem ao destino correto, seja um site, um serviço de streaming ou outra pessoa.
DNS
DNS, acrónimo de Domain Name System (Sistema de Nomes de Domínio), é a "agenda telefónica da Internet", que traduz nomes de domínio legíveis por humanos, como www.google.com, nos endereços IP numéricos que os computadores usam para se encontrar na rede. Sem o DNS, os utilizadores teriam de memorizar os endereços IP de cada site, o que não seria prático.
Roteador
Um roteador é um dispositivo que distribui o sinal da internet para múltiplos aparelhos dentro de uma rede, como computadores, smartphones e tablets, criando uma rede local (LAN) e permitindo que todos utilizem a mesma conexão sem fio (Wi-Fi) ou por cabo. Ele funciona recebendo dados do modem, determinando o destino correto e encaminhando os pacotes de dados para os dispositivos conectados, e também pode oferecer funções de segurança, como firewalls.
